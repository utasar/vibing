# Quick Conversion Reference

## Powers of 2
2^0 = 1     2^8 = 256
2^1 = 2     2^9 = 512
2^2 = 4     2^10 = 1024
2^3 = 8     2^11 = 2048
2^4 = 16    2^12 = 4096
2^5 = 32    2^13 = 8192
2^6 = 64    2^14 = 16384
2^7 = 128   2^15 = 32768
